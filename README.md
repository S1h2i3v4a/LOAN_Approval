# LOAN_Approval

# 🏦 Loan Approval Machine Learning Project

## 📌 Overview

Ye project ek Machine Learning pipeline hai jo predict karta hai ki loan approve hoga ya nahi based on applicant details.

Maine is project ko beginner level se start karke step-by-step ek complete ML pipeline me convert kiya — with proper structure, debugging, and improvements.

---

## ⚙️ Modules Breakdown

1. Project Setup & Data Loading
2. Data Cleaning (Missing Values Handling)
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Encoding (Categorical → Numerical)
6. Correlation Analysis
7. Train-Test Split
8. Feature Scaling
9. Model Training (Logistic, KNN, Naive Bayes)
10. Model Evaluation
11. Feature Engineering
12. Final Model Selection

---

## 🧠 Approach

Initially, maine directly model training start kar diya tha, but mujhe realize hua ki preprocessing aur pipeline me flaws the.

Isliye maine:

- Data cleaning improve ki
- Feature importance analyze ki
- Proper scaling aur encoding apply ki
- Multiple models compare kiye

Final me ek structured approach follow kiya instead of random experimentation.

---

## 🚧 Challenges Faced

- Missing values handling confusion
- Encoding strategy choose karna (Label vs OneHot)
- Model selection (accuracy vs precision tradeoff)
- Feature engineering ka correct use

---

## 💡 Key Learnings

- ML me data preprocessing sabse important hota hai
- Feature engineering se model performance improve hota hai
- Accuracy alone enough nahi hoti
- Structured pipeline banana zaroori hai

---

## 🏆 Final Result

- Best Model: Logistic Regression
- Accuracy: ~88%
- Balanced Precision & Recall
